Resumos de HTML e CSS

1. Elementos HTML

Elementos Inline: Ocupam apenas o espaço necessário para seu conteúdo e não iniciam uma nova linha. Eles podem ser colocados dentro de elementos block e inline.
Exemplos: span, a, strong, em, img, label, input; Uso: Para estilizar partes de um texto ou incluir elementos pequenos em uma linha.

Elementos Block: Ocupam toda a largura disponível e iniciam uma nova linha, empurrando os próximos elementos para baixo. Podem conter elementos block e inline.
Exemplos: div, p, h1 até h6, ul, ol, section, article; Uso: Para estruturar layouts e dividir o conteúdo em seções lógicas.

Elementos Inline-Block: Se comportam como elementos inline (ficam na mesma linha), mas permitem definir largura e altura, ao contrário dos elementos inline normais.
Exemplos: button, input (dependendo do estilo), img; Uso: Para criar elementos que devem manter propriedades de bloco, mas permanecer na mesma linha de outros elementos.

1.1 Tags Semânticas

  • header: Define o cabeçalho da página.
  • nav: Define uma seção de navegação.
  • main: Define o conteúdo principal.
  • article: Representa um conteúdo independente.
  • section: Agrupa elementos relacionados.
  • aside: Define conteúdo complementar.
  • footer: Define o rodapé.
  • figure: Representa um elemento de mídia.
  • figcaption: Fornece legenda para o elemento figure.
  • time: Representa uma data ou hora.
  • mark: Destaca partes do texto.
  • summary: Define o sumário de um elemento details.
